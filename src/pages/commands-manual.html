Linux Windows Descrição do Comando
cd
( change directory )		 cd
( change directory )		 Comando para mudar de pasta.
ls
( list )		 dir
( directory )		 Visualizar a estrutura de pastas do diretório que está ou que deseja.
mkdir
( make directory )		 md
( make directory )		 Cria pastas
rmdir
( remove directory )		 rd
( remove directory )		 Apaga pastas. No Linux, uma alternativa também é rm –rf.
clear cls
( clean screen )		 Limpa a tela da linha de comando, removendo todos os comandos já feitos até então.
cp
( copy )		 copy Copia arquivos e pastas.
man
( manual )		 help Ajuda para saber qual a função de um determinado comando desejado.
mv
( move )		 move Move um arquivo ou pasta de lugar, é como se fosse a opção de “cortar”, do Ctrl + X.
