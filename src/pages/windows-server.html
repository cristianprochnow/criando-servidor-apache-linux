Servidor Apache em uma Máquina Windows

Para realizar todo esse processo no Windows, é um pouco mais fácil. Já que o Windows não permite muito a facilidade em instalar pacotes e algo semelhante, que o Linux fornece, então usaremos uma aplicação para fazer esse papel. Contando que já estamos no Windows, então basta instalar o XAMPP, que é uma aplicação que simulará um servidor Apache na nossa máquina, poupando configurações e tempo.

Para isso, basta realizar o download para a versão de Windows, e ao baixar, instalar no esquema clássico de clicar em “Próximo”, “Próximo”, “Próximo” até concluir.

Após isso, acesse a pasta htdocs, dentro da pasta de instalação do Xampp, e coloque seus arquivos lá dentro. Ao rodar a aplicação, inicie o servidor Apache e então basta ir em qualquer navegador e pesquisar o endereço localhost. E pronto, estarão listados ali os arquivos de seu projeto.
